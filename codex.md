# MangaHub

Personal manga/manhwa tracking app.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- Supabase
- MangaUpdates API

## Current Features

- User authentication
- Search MangaUpdates
- Add series to library
- Reading / Completed / On Hold / Dropped / Plan to Read
- Chapter progress tracking
- Cover image support
- Duplicate prevention

## Roadmap

1. Series detail pages
2. Library search
3. Schedule improvements
4. Import/export
5. Better mobile UI

## Notes

- Use MangaUpdates as primary metadata source.
- Prefer TypeScript types over `any`.
- Keep UI mobile-first.
